# 可能干扰内核模块的常见应用程序

由于手动映射内核驱动程序会受到许多应用程序的限制（包括一些看似无关的应用程序），了解潜在的干扰情况非常重要。以下是一些可能干扰驱动映射的常用应用程序列表，在映射驱动之前应关闭或停止这些程序：

- 常见带有反作弊的游戏
  - 带有 Facit 或 ESEA 的 CS:GO / CS2
  - EAC（Apex、Fortnite、Rust）
  - Vanguard（Valorant）
- 防病毒应用程序
- Windows Defender

:::warning
请*不要*永久禁用您的防病毒软件。  
仅在映射驱动时临时禁用！
:::

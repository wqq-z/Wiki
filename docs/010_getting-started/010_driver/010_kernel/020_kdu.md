# 使用 KDU 加载 Valthrun 内核驱动

## 什么是 KDU

KDU 是一个多功能工具，旨在简化对 Windows 内核组件的探索，避免了繁琐的设置或本地调试器需求。其主要功能之一是驱动加载器，可以绕过驱动签名强制 (DSE)，从而更轻松地加载未签名的驱动程序，如 Valthrun 内核驱动。

## 加载 Valthrun 内核驱动

要使用 KDU 加载 Valthrun 内核驱动，请参阅官方 KDU GitHub 仓库中的详细说明：[KDU GitHub Repository](https://github.com/hfiref0x/KDU)。该资源提供了关于使用 KDU 驱动加载功能的全面指导，包括成功执行所需的任何前提条件和使用步骤。
